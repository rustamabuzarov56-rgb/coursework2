# JobSearcher
Простое приложение для поиска и анализа вакансий с сайта HeadHunter. 
Оно предоставляет удобный способ поиска вакансий по ключевым словам, 
их хранение и просмотр в структурированном виде.

## Описание
Приложение построено на основе абстракций для взаимодействия с API сервисов поиска вакансий и
хранит собранные данные в локальном JSON-файле. Основной целью является предоставление простого способа сбора и
просмотра вакансий без необходимости постоянного обращения к веб-сайту.

## Установка

1. Клонируйте репозиторий:

```git clone https://github.com/yourusername/jobsearcher.git```

   
2. Установите зависимости:

```poetry install```


## Использование
Запустите приложение следующим образом:

```python main.py```

Программа предложит ввести поисковый запрос и покажет подходящие вакансии. Вы можете сохранить интересующие вакансии в
файл для последующего просмотра.

## Структура проекта

```main.py```: точка входа в приложение.
```api_module.py```: модуль для работы с API HeadHunter.
```file_module.py```: работа с JSON-файлами для хранения данных.
```vacancy_module.py```: модуль для работы с вакансиями

# Требования
Python >= 3.7
Библиотека Requests для HTTP-запросов.
